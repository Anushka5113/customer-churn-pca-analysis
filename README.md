This project applies Principal Component Analysis (PCA) to a Customer Churn dataset to reduce the dimensionality of customer information while preserving the most important patterns in the data. The dataset includes features such as age, annual income, dependents, tenure, senior citizen status, monthly charges, total charges, number of services, phone service, internet service, and churn. The project involves data preprocessing, handling missing values, encoding categorical variables, and feature scaling before applying PCA. The transformed data is then visualized to understand customer distributions and identify patterns related to customer churn. Developed using Python, Pandas, NumPy, Scikit-learn, Matplotlib, and Seaborn, this project demonstrates how PCA can simplify high-dimensional datasets, improve data visualization, and prepare features for future machine learning models such as Logistic Regression, Random Forest, or Support Vector Machine.
